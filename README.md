# imeshameegoda.github.io

This is a personal website project. Below you'll find an overview, setup instructions, and project structure.

## Overview
This repository contains the source code for a personal or portfolio website. It includes HTML, CSS, and JavaScript files for interactive and animated web content.

## Features
- Responsive design
- Custom animations
- Interactive navigation
- Social links integration

## Project Structure
- `index.html` — Main landing page
- `style.css` — Main stylesheet
- `animations.css` — Animation-specific styles
- `nav.js` — Navigation bar logic
- `cursor.js` — Custom cursor effects
- `reveal.js` — Reveal-on-scroll animations
- `scroll.js` — Scroll-based effects
- `social-links.json` — Social media links data
- `intergram-widget.html` — Chat widget integration

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/imeshameegoda/imeshameegoda.github.io.git
   ```
2. Open `index.html` in your browser to view the site locally.
3. Edit the HTML, CSS, or JS files to customize your site.

## Deployment
This site is designed for GitHub Pages. To deploy:
- Push changes to the `main` branch.
- Visit `https://imeshameegoda.github.io/` to see your live site.
